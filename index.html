My Portfolio Some Of My Work

Image Editor

The Streamlit Image Editor App is a Python-based application that enables users to upload and edit images through a user-friendly interface. It leverages the Streamlit library alongside PIL, OpenCV, and NumPy for a range of editing features like resizing, rotation, filters, color adjustments, and cropping. Users can undo changes and download the edited image. The app includes files for editing functions, image uploading, and main app configuration. It's customizable and primarily educational, providing a basis for more complex image editing solutions.

Expense Tracker

The Streamlit Expense Tracker is a web application designed for managing personal finances, tracking expenses and incomes, and visualizing financial data. It utilizes Streamlit for a responsive and intuitive user interface, offering features like user authentication, categorized expense tracking, data visualization through Sankey diagrams, and a monthly tracker for recurring expenses. The application is modular with dedicated scripts for app logic, authentication, database management, and other functions. It emphasizes security, especially in data handling and user information, and encourages contributions through a standard open-source workflow.

Ticket-X

The Django Ticket-X is a web application designed for managing IT tickets, offering a seamless platform for users to register and manipulate records. It utilizes Django for a robust and efficient backend, providing features like user authentication, record management, and customizable forms for various user interactions. The system boasts a clean interface for adding, viewing, updating, or deleting records, and integrates a Record model for streamlined ticket administration. Ticket-X is structured with clear modules for different functionalities including views, URLs, and model management, focusing on efficiency and user-friendliness. It encourages a productive workflow for managing IT-related records and tasks.

Product Price Tracker

The Streamlit PPT is a specialized application for tracking and analyzing product prices on the eMag retail platform. It's developed using Python and employs a modular design that includes web scraping, data storage, and user interfaces via Streamlit. The application is comprised of various scripts dedicated to user interactions, data handling, web navigation using Selenium, and configuration management.
